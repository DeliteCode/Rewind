                            //reverse the string
                            function reverseString(userString) {
                                let revString = [];
                                for (let index = userString.length - 1; index >= 0; index--) {
                                    revString += userString[index];

                                }
                                return revString;
                            }                           
                        
                    



                                                
                            //display the reversed string
                            function displayString(revString) {
                                document.getElementById("msg").innerHTML = `Your string reversed is: ${revString}`;
                                document.getElementById("alert").classList.remove("invisible");
                            }

The Code is structed in three functions

getValue:

The function getValue() will acess the element of the web page identified by the "alert" id, setting them to "invisible". Then it will get the value of the element of the page identified by the "userString" id, assigning it to a variable called userString. The content of this variable will be provided as argument to the function reverseString(), and the return of this function will be assigned to a variable called "revString". Variable "revString" will then be provided as argument to the function displayString(), to be displayed at the screen.

reverseString:

Function reverseString() will work with one parameter called "userString", which in this case will contain the string we want to reverse.The first line of this function will declare an empty array called "revString" to be the new container of the reversed string. Then, it will loop the this parameter, "userString" backwards, rebuilding this string into the the array "revString", character by character, now backwards, returning the content of this array.

displayString:

Function displayString will contain one parameter, "revString" which contain the return of the function "reverString", displaying this content at the element of the HTML document identified as "msg", finally turning the HTML element identified as "alert" to visible, allowing the user to see what is displayed.
